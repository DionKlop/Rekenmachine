# Rekenmachine

Deze rekenmachine ondersteunt de volgende bewerkingen:

* Optellen
* Aftrekken
* Vermenigvuldigen
* Delen
* Ophogen (+1)
* Verlagen (-1)
* Verdubbelen (×2)
* Halveren (÷2)

Na elke berekening wordt de uitkomst gebruikt als eerste getal (`n1`) voor een volgende berekening.

## Variabelen

* **n1** = het eerste ingevoerde getal of de uitkomst van de vorige berekening.
* **n2** = het tweede ingevoerde getal.

## Bewerkingen met `n1` en `n2`

| Bewerking        | Formule   |
| ---------------- | --------- |
| Optellen         | `n1 + n2` |
| Aftrekken        | `n1 - n2` |
| Vermenigvuldigen | `n1 * n2` |
| Delen            | `n1 / n2` |

## Bewerkingen met alleen `n1`

| Bewerking   | Formule  |
| ----------- | -------- |
| Ophogen     | `n1 + 1` |
| Verlagen    | `n1 - 1` |
| Verdubbelen | `n1 * 2` |
| Halveren    | `n1 / 2` |
